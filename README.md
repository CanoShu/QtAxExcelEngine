# QtAxExcelEngine

July 17, 2016 4:38 PM
QtAxExcelEngine 是一个使用Qt's ActiveX(OLE) QAxObject 读写EXCEL 的辅助类，同时可以和Qt tableWidget 做交互。
既然是使用OLE,当然你必须先安装EXCEL。才能实用这个工具类。
